## Summary
Added `blockchainTxHash` to the `gifts` schema in PostgreSQL (`src/lib/db/schema.ts`) and SQLite (`src/server/db/schema.ts`) along with the required Drizzle indexes. Updated both internal and public gift confirmation API routes to extract and securely save the blockchain transaction hash from the client request payload after Soroban contract calls conclude. This facilitates linking directly to the Stellar block explorer from within the gift summaries.
